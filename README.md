# USD TO RUB

**Very simple dollar to ruble converter.**

---

# Requirements

* Python (3.6, 3.7)

# Installation

Install using `python`...

    python setup.py install

# Example

Let's take a look at a quick example of using USD TO RUB converter.

Install and use like so...

    python setup.py install
    usd-to-rub 1000

That's it, we're done!
